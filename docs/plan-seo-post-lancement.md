## 🔄 En continu

- **Cohérence NAP** : Nom / Adresse / Téléphone strictement identiques sur le site, la fiche Google, les annuaires et les réseaux. C'est un signal de confiance majeur.
- **Photos fraîches** sur la fiche Google (Google adore) et sur le site — remplacer les visuels provisoires par les photos définitives dès qu'elles existent (alambic, chais, paysages de Moindou).
- **Actualités** : à chaque nouveauté (whisky au gaïac, brandy d'hydromel, médaille, salon), un post GBP + réseaux. Le jour où le site aura une section actualités, chaque sortie de cuvée devient une page indexable.
- **Suivi mensuel** (10 min) : Search Console → requêtes qui montent, pages indexées ; fiche Google → appels, itinéraires, vues.
